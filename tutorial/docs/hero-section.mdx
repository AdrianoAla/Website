## Main element

### Accepted elements before header

 - Background image *(optional)*
    - Takes an image element.
    - If none is provided the background will be the primary color defined by the user in the web theme.

### Accepted elements in the header
- Subheading
- Title
- Subtitle

### Additional elements in body
  - Newsletter *(optional)*
    - Takes a paragraph element as a prompt.
  - Video *(optional)*
    - Takes a thumbnail and a video element.
  - Action Link *(optional)*
    - Takes a link element.
